# 客服自动回复

这是一个简单的网页客服自动回复演示项目，包含一个可展开的聊天窗口和自动回复功能。

## 文件说明

- `index.html`：主页面，显示客服聊天窗口与自动回复逻辑。
- `static/chat.css`：聊天页面样式。
- `static/iconfont.css`：图标字体样式。
- `static/jquery-3.4.1.min.js`：jQuery 库，用于 DOM 操作和事件处理。

## 功能

- 打开/关闭客服聊天窗口
- 列表切换至聊天界面
- 输入文本后发送并显示用户消息
- 自动回复关键词触发响应

## 使用方法

1. 将整个项目目录放到 Web 服务器根目录下（如 Apache、Nginx、或本地静态服务器）。
2. 在浏览器中打开 `index.html`。
3. 点击聊天按钮展开对话窗口，输入文字并点击发送查看自动回复。

## 注意

- 当前示例仅做本地演示，自动回复规则写在页面内的 JavaScript 中。
- 若需生产环境使用，建议拆分样式和脚本，并使用后端接口实现自动回复逻辑。
